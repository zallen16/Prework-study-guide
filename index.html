HTML

  • It is made up of elements which work together to provide content and the basic framework for all pages on the web.
  • HTML elements work together to build the structure of a webpage and provide a framework that makes it easy for a user to navigate the webpage.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is the bone structure of the site if we're comparing everything to the human body.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A border indicates how much space we want in between the margin and the padding.
  • The order in which these elements appear from outside inward are Margin, Border, Padding, and Content.

Git

  • It is a version control system that lets you manage and keep track of each version of your project.
  • Git commands always start with "git" and are used to save and track versions of your code.
  • git status: checks what branch we are currently on.
  • git checkout -b branch name: creates a new branch and switches to it.

JavaScript

  • A high-level dynamic programming language that allows users to interact with websites.
  • You can use it to power your websites with effects, interactivity, data communication, and much more.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
